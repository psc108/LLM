Terraform Setup Required

Terraform CLI is not installed

The Terraform sandbox requires the Terraform CLI to be installed on your system. Please follow the instructions below to install it:

Installation Instructions:

Choose the instructions based on your operating system:

Linux:
wget -O- https://apt.releases.hashicorp.com/gpg | gpg --dearmor | sudo tee /usr/share/keyrings/hashicorp-archive-keyring.gpg
echo "deb [signed-by=/usr/share/keyrings/hashicorp-archive-keyring.gpg] https://apt.releases.hashicorp.com $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/hashicorp.list
sudo apt update && sudo apt install terraform
macOS (with Homebrew):
brew tap hashicorp/tap
brew install hashicorp/tap/terraform
Windows (with Chocolatey):
choco install terraform

After installation, verify by running terraform --version in your terminal.

Once Terraform is installed, refresh this page to continue using the Terraform sandbox.
